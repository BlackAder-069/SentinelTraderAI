# SentinelTraderAI

This project is an agentic trading system with Streamlit frontend, LangChain + Gemini agent, and Alpaca paper trading.

## Setup

1. Create a virtual env and activate
2. pip install -r requirements.txt
3. Copy `.env.example` to `.env` and fill keys

## Run Streamlit UI

streamlit run streamlit/app.py

## Run pipeline

python main.py

## Tests

pytest


Examples which can be used in frontend are - AAPL, MSFT, GOOGL, AMZN, TSLA, NVDA, SPY, QQQ
